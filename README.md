## Getting Started

Preview: https://star-wars-template.vercel.app/

First, install the dependencies:

```bash
yarn
# or
npm install
```

Run the development server:

```bash
yarn dev
# or
npm run dev
```

Run tests:

```bash
yarn test
# or
npm run test
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
